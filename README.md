## Why Linear Models?

Linear regression is the foundation of supervised ML:
- Simple yet powerful
- Fully interpretable
- Reveals data structure
- Teaches diagnostic thinking
- Serves as baseline for all advanced models

Understanding linear failure modes is critical before moving to tree-based or deep models.
